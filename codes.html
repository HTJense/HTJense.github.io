CosmoPower

CosmoPower is a machine learning-accelerated bayesian inference library for python. It aims to be a fully differentiable library for cosmological analyses, providing neural network emulators of power spectra.

Link: please refer to the original cosmopower page for the time being.


pyWMAP: a WMAP likelihood implementation for Cobaya

pyWMAP is a WMAP Likelihood implementation for Cobaya. The code is a python adaptation of the high-ell likelihood components of the publicly available WMAP v5 likelihood, retaining only data in temperature and high-l polarization.

Link: github page.
