pyWMAP

pyWMAP is a new python implementation of the original Fortran WMAP v5 likelihood. It is designed to work with the cobaya sampling software while replicating as much as possible the original likelihood's behaviour under similar settings.

pyWMAP can derive the cosmological parameters and Sunyaev-Zel'dovich effect intensity from the original WMAP temperature and high-l polarization data.

You can download pyWMAP here.

Installation

To download and install the likelihood, see the following example commands:

	~$ git clone https://github.com/HTJense/pyWMAP.git
	~$ cd pyWMAP
	~$ pip install -e .
	~$ cobaya-install wmaplike.WMAPLike

The first two commands will download and install pyWMAP and navigate to the newly created directory. Then we install the package dependencies using pip, and finally we install the likelihood with cobaya. This last command should download the WMAP v5 data from the NASA LAMBDA archives. Note that this does not download the SZ template included in WMAP, but the likelihood will check if the template exists and download it at a later point if it needs it.

If for any reason either of the last two commands does not work, you can alternatively invoke pip and cobaya from python with for example:

	~$ python3 -m pip install -e .
	~$ python3 -m cobaya install wmaplike.WMAPLike

You can afterwards test pyWMAP by running

 ~$ cobaya-run eval_wmap.yaml

Which should run without errors.

Included in the git repo is a test_wmap.py script that can be run to evaluate the different components of the likelihood and check to see if any of them disagrees with the original WMAP v5 likelihood.
